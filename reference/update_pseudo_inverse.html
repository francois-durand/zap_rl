update_pseudo_inverse

zap_rl.update_pseudo_inverse(array_a, vector_c, vector_d, array_a_dagger)[source]

Update the pseudo-inverse of a matrix of real numbers.

Bibliography: “Generalized Inversion of Modified Matrices”, Meyer, 1973.

Parameters

  • array_a (ndarray) – Rectangular numpy array.

  • vector_c (ndarray) – Numpy array (vector). Length = number of rows of array_a.

  • vector_d (ndarray) – Numpy array (vector). Length = number of columns of array_a.

  • array_a_dagger (ndarray) – Pseudo-inverse of array_a (same shape as array_a.T).

Returns

array_a_dagger is updated to become the pseudo-inverse of array_a + np.outer(vector_c, vector_d).

Return type

None

Notes

All arrays must have real coefficients (not complex numbers).
